# History Reconstruction Notice

The original local Git metadata for development before v0.4.0-alpha was
accidentally deleted on 14 July 2026.

The source code itself was recovered from archived milestone ZIP packages.
The earlier commits and tags in this repository were reconstructed from those
snapshots to preserve a transparent technical audit trail.

The reconstructed history does not reproduce the original commit hashes,
timestamps, authorship sequence, intermediate commits, or branch topology.

Each reconstructed commit is explicitly labelled as a source snapshot.

v0.4.0-alpha was independently restored, built, deployed, tested, and confirmed
working before reconstruction began.

No separate v0.3.0 source archive survived. Its changes remain documented in
CHANGELOG.md, but no artificial v0.3.0 Git snapshot has been created.
